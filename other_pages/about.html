I recently took a programming langauges class! After dealing with lambda calculus and haskell, it really makes you miss convential programming langauges like C, C++, C#, and Java! Anyways, here is one of the functions I made in haskell! 

                
                    assoc :: Int -> String -> [(String, Int)] -> Int
                    -- if the recursive result leads to the empty list, return the default value
                    assoc def key [] = def
                    -- recurse on the tail, while pattern matching on (s, x) which is the head
                    assoc def key ((s,x):xs)
                        -- if the key is the same as the string in the pair, return the pairs value
                        | key == s = x
                        -- else recurse on the tail
                        | otherwise = assoc def key xs

this tail recursive function searches the list for the first i such that ki is equals to key. If such a ki is found, then vi is returned. Otherwise, if no such ki exists in the list, `def` is returned. Overall, haskell can be a bit tought to understand.
